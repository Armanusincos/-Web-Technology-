7. Оператор присваивания

Оператор присваивания служит для записи значения выражения в переменную. Общий вид:

имя_переменной = выражение

Примеры:

  • A = 5
  • B = A + 10
  • C = B / 2

8. Форматированный ввод и вывод

Для ввода и вывода данных в Фортране применяются операторы READ, WRITE и PRINT.

Оператор Назначение
READ Ввод данных
WRITE Вывод данных
PRINT Упрощённый вывод на экран

Форматированный вывод позволяет задавать, как именно будут отображаться числа, строки и другие данные.

9. Пример программы

Ниже приведён простой пример программы на Фортране: 

PROGRAM HELLO
    PRINT *, 'HELLO, WORLD!'
END

Эта программа выводит на экран сообщение HELLO, WORLD!.

Краткие выводы

  1. Фортран предназначен прежде всего для вычислительных задач.
  2. Программа имеет строгую структуру.
  3. Язык поддерживает основные типы данных и арифметические операции.
  4. Для работы с данными применяются операторы присваивания, ввода и вывода.
