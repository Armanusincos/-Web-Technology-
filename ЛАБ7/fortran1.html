Фортран: разделы 1–6

1. Общие сведения о языке Фортран

Фортран — один из первых языков программирования высокого уровня. Он был разработан для научных, инженерных и математических вычислений. Название FORTRAN происходит от слов FORmula TRANslation.

2. Алфавит языка

Алфавит языка включает буквы латинского алфавита, цифры и специальные символы, используемые для записи операторов, выражений и служебных конструкций.

2.1. Буквы и цифры

В именах переменных и команд могут использоваться латинские буквы от A до Z и цифры от 0 до 9.

2.2. Специальные символы

В языке Фортран используются следующие специальные символы:

Символ Назначение
+ Сложение
- Вычитание
* Умножение
/ Деление
= Присваивание
( ) Скобки

3. Структура программы

Программа на Фортране состоит из заголовка, описаний переменных, исполняемых операторов и завершения программы.

3.1. Заголовок программы

Начало программы обычно задаётся словом PROGRAM, после которого следует имя программы.

3.2. Операторы программы

После заголовка размещаются операторы описания и исполняемые операторы. Завершается программа словом END.

4. Типы данных

В Фортране используются различные типы данных:

  • INTEGER — целые числа
  • REAL — вещественные числа
  • DOUBLE PRECISION — числа двойной точности
  • COMPLEX — комплексные числа
  • LOGICAL — логические значения
  • CHARACTER — символьные данные

5. Переменные и константы

Переменные используются для хранения данных, которые могут изменяться во время выполнения программы. Константы представляют собой неизменяемые значения.

Примеры имён переменных:

  • A
  • X1
  • SUMMA
  • TEMP

6. Арифметические выражения

Арифметические выражения в Фортране строятся с использованием чисел, переменных, операций и скобок.

Выражение Смысл
A + B Сумма A и B
A - B Разность A и B
A * B Произведение A и B
A / B Частное A и B
(A + B) * C Сумма A и B, умноженная на C
